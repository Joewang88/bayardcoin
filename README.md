## How to add another product

1. Upload product images to the matching category folder.
2. Name images using the global item number:
   - 00002-1.jpg
   - 00002-2.jpg
   - 00002-3.jpg
3. Open products.js.
4. Copy an existing product block and change item number, title, price, status, grade, cert link, images and description.
